Project Description and Raw Data

A.	Overview

This folder contains the original project documentation and raw dataset used for the Bright Coffee Shop Dashboard project. These files form the foundation for the data analysis and visualisation process.

B.	Contents

1.	Project Description

1.1.	A PDF document outlining the project requirements, objectives, and expected deliverables
1.2.	Provides context for the dashboard development and analysis

2.	Raw Dataset

2.1.	Original dataset in CSV format
2.2.	Contains transactional sales data used for analysis
2.3.	Includes fields such as transaction date, product name, quantity, unit price, and store location

C.	Usage
1.	The raw dataset was used for data cleaning and transformation.
2.	No modifications were made to the original dataset in this folder
3.	Processed data is stored separately to maintain data integrity

D.	Note

This folder contains raw and unprocessed data.
All transformations and analysis were performed on a cleaned version of the dataset.

E.	Purpose

The purpose of this folder is to:

1.	Maintain a clear record of original data sources
2.	Ensure transparency in the data analysis process
3.	Support reproducibility of the project
