Project Planning

A. Overview

This folder contains project planning materials used to guide the development of the Bright Coffee Shop Dashboard. These documents outline the project structure, workflow, and timeline.

B. Contents

1. Miro Diagram
   
1.1. Visual representation of the project workflow and data process
1.2. Illustrates key steps from data collection to dashboard creation

2. Gantt Chart

2.1. Timeline of project tasks and milestones
2.2. Shows project phases, dependencies, and progress tracking

C. Purpose

1. To plan and organise the project effectively
2. To visualise the workflow and development process
3. To ensure timely completion of all project tasks

D. Usage

These documents were used as a guide throughout the project to:

1. Track progress
2. Maintain structure and direction
3. Align tasks with project objectives
